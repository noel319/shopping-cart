# Shopping cart

A simple fake store React app.

[Live Demo](https://michalosman.github.io/shopping-cart/) :point_left:

## Getting started

```
git clone https://github.com/michalosman/shopping-cart.git
cd shopping-cart
npm install
npm start
```
